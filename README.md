# Klingnum

> Returns a number in Klingon text


## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i klingnum
```

## Usage

```js
klingnum(number)
``` 


Where `number` is a regular number ranging from 0 to 9,999,999.
(No punctuation, spaces, leading zeros, or alphabetic characters allowed)
